# Contributing to grandMA3-Chataigne-Bridge

Thank you for your interest in contributing to this project! To keep this bridge stable for live show environments, please follow these guidelines.

## Reporting Issues
If you find a bug, please use the **Issue Tracker**. Before submitting, ensure that:
* You are using the latest version of the plugin.
* You check if the issue has already been reported.
* **Crucial:** Always state which version of grandMA3 you are running (e.g., 2.4.2).

## Pull Requests
1. Fork the repository.
2. Create your feature branch (`git checkout -b feature/amazing-feature`).
3. Commit your changes.
4. Push to the branch (`git push origin feature/amazing-feature`).
5. Open a Pull Request.

Please keep the code clean, documented, and ensure it passes basic testing on your console before submitting.
